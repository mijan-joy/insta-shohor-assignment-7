Local Storage and Session Storage Difference

Local Storage

1. Storage size 5mb - 10mb

2. Data doesn't exipire after a session ends.

3. Data doesn't disapper until the user deletes it.

4. Sometimes harmful for security issues.

5. It works all tab in browser

Session Storage

1. Storage size 5mb

2. Data disappears after a session ends.

3. Each new tab creates a new session for the same URL.

4. If a tab is copied to a new tab the session storage is also copied.

5. I works selected tab. When tab close session data will deleted
